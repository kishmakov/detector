# About

Replication and exploration of https://arxiv.org/abs/2306.04723 — AI-text detection via intrinsic dimensionality of token embeddings.

- `main_paper_src/` — LaTeX sources of the original paper
- `main_paper_data/` — code and data from the original paper
- `magnitude.md` — notes on magnitude-based dimension estimation
- `papers/` — related work (GLTR, Binoculars, RAID, ZeroGPT, MGTBench)

# Data

`data/completions.db` — SQLite database of LLM completions (human + AI), created by `scripts/db/01_init.py` and filled by `scripts/db/01_fill.py`. Schema in `data/migrations/01_init.sql`.

`scripts/add_completion.py` — script to insert completions generated by new LLMs.
# Source (`src/`)

| File | Purpose |
|---|---|
| `embeddings_provider.py` | Load HuggingFace models; compute token embeddings |
| `embeddings_dataset.py` | PyTorch Dataset wrapping DB iterator |
| `db_iterator.py` | Iterate over completions gathered from `completions.db` |
| `magnitude.py` | Magnitude function for intrinsic dimension estimation |
| `logistic_regression.py` | Tools for train/eval logistic regression classifier |
| `text_utils.py` | Text preprocessing helpers |

# Examples (`examples/`)

- `test_magnitude.py` — test magnitude-based detection
- `test_simple_logreg.py` — test logistic regression detection
- `make_plots.py` — generate per-text embedding plots
- `texts/` / `plots/` — sample texts and their output plots
- `runs.md` — log of experiment runs

# Application (optional, for live demo)

## Config

`config.sh` — stores server IP. After editing, run `bash chrome/scripts/build.sh` to regenerate `chrome/config.js`.

| Variable | Description |
|---|---|
| `DETECTOR_BASE_URL` | Base URL of deployed service (used by live tests) |

## `server/` — FastAPI backend

- **Stack:** Python 3.11, FastAPI, Uvicorn, Pydantic v2
- **Run:** `docker-compose up` or `uvicorn app.main:app --reload` inside `server/.venv/`
- **Test (local):** `.venv/bin/pytest tests/test_analyze.py -v`
- **Test (live):** `bash scripts/test_live.sh`
- **Deploy:** `bash scripts/deploy.sh` — rsync + Docker build on remote host `GC`

API: `GET /health` → `{"status":"ok"}`, `POST /analyze` `{"text":"..."}` → `{"word_count":N}`

## `chrome/` — browser extension

Load unpacked from `chrome://extensions`. Right-click selected text → **Analyze** → check service worker console for result.
